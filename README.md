# react-spring-boiler-plate

리액트와 스프링 프로젝트를 위한 보일러플레이트입니다

**Front** React with TS, styled-component

- devtools
  - babel
  - eslint
  - prettier
  - tsconfig
  - webpack

**Back** Spring Boot(2.2.5)

- dependencies:
  - devtools
  - JPA
  - mysql
  - web
  - Lombok
    - need install plugin in VSCode or IntelliJ
    - enable lombok annotation processor in IntelliJ
  - JUnit4
